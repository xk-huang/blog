概念

Tmux是一个 BSD 协议发布的终端复用软件，用来在服务器端托管同时运行的 Shell。这意味着一些需要实时在线的服务（如SSH，10min无数据流通自动断线）可通过Tmux的session保留工作空间。每个Session含有若干window，每个window可分为若干Pane。
